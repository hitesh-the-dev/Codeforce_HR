Hard & Tricky Level Questions

Question 1

What is the output of the following code?

int arr[3] = {10, 20, 30};
int *p = arr;
printf("%d ", *p);
printf("%d ", *p++);
printf("%d ", *++p);

Question 2

What does the following code print?

int arr[3][3] = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
int *p = &arr[1][1];
printf("%d %d %d", *(p-1), *p, *(p+3));
